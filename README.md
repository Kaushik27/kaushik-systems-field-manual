# Kaushik Mandal — Systems Field Manual

A single-page editorial portfolio for Kaushik Mandal, built around verified material from his existing portfolio, LinkedIn profile snapshot, and public GitHub profile.

## What is included

- Responsive portfolio with keyboard-friendly navigation and disclosures
- Selected professional systems, full career trace, technology index, public lab, and credential ledger
- Copy-email interaction with a mail fallback
- Person structured data and host-aware Open Graph / X metadata
- Bespoke social preview card in `public/og.png`
- Reduced-motion and visible-focus accessibility support

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run build
node --test tests/rendered-html.test.mjs
```
